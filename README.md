# webpack-babel-es6

Example modern JavaScript project using:

* Webpack 4.41.x
* Babel 7.7.x
* Scss 4.13.x


## Installation

Install dependencies using:

    npm install


## Usage

Run the dev server using:

    npm start

Then view the local server at:

    http://localhost:8080


## Directory structure

    /                                 --> Project root
    /src                              --> Frontend sources files
